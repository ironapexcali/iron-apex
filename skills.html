Skill Builder Program

8 Week Intermediate Skill Development

Weeks 1–2

Monday — Pull

Pull-ups 5x8
Front Lever Tuck Hold 5 sets
Rows 4x12

Tuesday — Push

Dips 5x10
Decline Push-ups 4x20
Plank 3x90s

Friday — Handstand

Wall Handstand Hold 6 sets
Handstand Kick Ups 10 attempts

Weeks 3–4

Increase reps on main exercises

Weeks 5–6

Add one extra set to skill work

Weeks 7–8

Focus on controlled holds and clean technique
