Intermediate Calisthenics Program

8 week program designed to build explosive pulling strength, muscle ups and freestyle foundations.

Weeks 1-2 — Strength Foundation

Monday — Explosive Pull

Warmup
Dead Hang 3x40s
Scapula Pull Ups 3x12
Explosive Power
High Pull Ups 5x3
Chest to Bar Pull Ups 4x5
Skill Work
Muscle Up Attempts 6 sets
Band Assisted Muscle Ups Strength
Weighted Pull Ups 4x5 Core
Hanging Leg Raises 4x15

Tuesday — Push Strength

Warmup
Wrist Mobility
Shoulder Mobility Planche Prep
Planche Lean 5 sets
Pseudo Push Ups 4x10 Strength
Dips 5x8
Pike Push Ups 4x12 Skill
Wall Handstand Holds 5 sets

Wednesday — Freestyle Basics

Bar Swings 10 sets
Beat Swing Practice
Bar Turn Drills Core Work
L Sit 5 sets
Hollow Hold 4x40s

Thursday — Lever Strength

Tuck Front Lever Hold 6 sets
Front Lever Raises 4x6 Pull Ups 4x8
Rows 4x15

Friday — Conditioning

Push Ups 4x30
Dips 4x12
Circuit 3 rounds
Pull Ups x8
Push Ups x25
Squats x25

Weeks 3-4 — Increased Volume

Increase pull up weight and muscle up attempts. Add one extra set to all main exercises.

Weeks 5-6 — Skill Focus

Focus on clean muscle ups and longer front lever holds. Practice freestyle swings for 15-20 minutes each session.

Weeks 7-8 — Peak Strength

Weighted Pull Ups 6x4
Muscle Ups 6x3
Front Lever Hold 8 sets
Conditioning circuits increased to 5 rounds.

Saturday — Rest

Light stretching and mobility work.

Sunday — Rest

Full recovery.
