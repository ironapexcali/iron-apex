Beginner Calisthenics Program

8 week foundation program to build strength, mobility and bar control.

Weeks 1-2 — Basic Strength

Monday — Pull Strength

Dead Hang 3x30s
Scapula Pull Ups 3x10
Pull Ups 5x5
Australian Rows 4x12
Hanging Knee Raises 4x12

Tuesday — Push Strength

Push Ups 5x20
Dips 4x8
Pike Push Ups 4x10
Wall Handstand Practice

Wednesday — Core + Mobility

Hollow Hold 4x30s
L Sit Attempts 6 sets
Pancake Stretch
Shoulder Mobility

Thursday — Pull Volume

Pull Ups 4x6
Rows 4x15
Hanging Knee Raises 4x15

Friday — Push Volume

Push Ups 4x25
Dips 4x10
Handstand Practice 10 min

Weeks 3-4 — Skill Development

Add basic bar swings and increase pull up volume.

Weeks 5-6 — Strength Progression

Increase pull ups and dips by 2 reps each set.

Weeks 7-8 — Consolidation

Focus on perfect form and longer hangs.

Saturday — Rest

Stretching

Sunday — Rest

Recovery
