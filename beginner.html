Beginner Calisthenics Program

8 Week Strength Foundation Program

Weeks 1–2

Monday — Pull

Pull-ups 4x5
Rows 4x10
Hanging Knee Raises 4x12

Tuesday — Push

Push-ups 4x15
Dips 4x8
Plank 3x60s

Weeks 3–4

Increase reps by 2 each set

Weeks 5–6

Add one extra set to main exercises

Weeks 7–8

Focus on clean technique and maximum reps
